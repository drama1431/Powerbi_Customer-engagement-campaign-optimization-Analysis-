# Customer Engagement & Campaign Optimization Analysis

## Project Overview
This Power BI dashboard analyzes customer engagement and campaign performance to identify factors influencing campaign acceptance.

## Key Metrics
- Total Customers: 580
- Campaign Acceptance Rate: 10.34%
- Avg Customer Spending: 608.39

## Business Insights
- High-spending customers show higher campaign response.
- Website purchases positively influence campaign acceptance.
- Customers with complaints show lower conversion rates.

## Tools Used
- Power BI
- SQL
- Excel/CSV Dataset
